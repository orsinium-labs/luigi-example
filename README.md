
## Usage

1. Build containers:
    ```bash
    sudo docker-compose build
    ```
1. Run scheduler:
    ```bash
    sudo docker-compose up -d lx-luigid
    ```
1. Run tasks:
    ```bash
    sudo docker-compose up lx-project
    ```
1. Open [127.0.0.1:8082](http://127.0.0.1:8082/static/visualiser/index.html)
